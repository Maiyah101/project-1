## Project Overview

The purpose of my project is to create an interactive digital storytelling experience using HTML, CSS, and JavaScript. The webpage functions like a digital children’s storybook that tells a personal and emotional story through images, text, animations, sound, and user interaction.

The story is based on a dream vacation shared between me and my grandfather. Each page represents a moment from the imagined trip. As users click through the pages, the story gradually shifts from joyful memories to the emotional realization that the experience only exists as a dream because my grandfather has passed away.

Key interactive elements include:

Clicking to flip through story pages

Background music that plays during interaction

Animated transitions between pages

Visual effects that enhance emotional storytelling

The interaction controls the pacing of the story and allows users to experience it step by step.
## Features
my project includes multiple CSS animations:

Fade-In Animation – Each page smoothly fades in when it becomes active.

Bounce Animation – The title on the cover page gently bounces to create attention and movement.

Zoom Animation – Images slightly zoom in when a page becomes active to create depth and visual interest.

These animations make the story feel dynamic and visually engaging instead of static.
